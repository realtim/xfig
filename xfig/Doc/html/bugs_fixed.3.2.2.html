Bugs Fixed in xfig 3.2.2

(Released Jul 2, 1998)

Placing a library object caused xfig to die after recursing for a long time

Arrowheads in xfig 2.x files were incorrectly converted to 3.2.1 (they came out too large)

Imakefile didn't properly install Electrical/Physical and Electrical/Schematic directories

Some variables multiply defined in u_draw.c

Some systems' "install" program don't have the "-d" (create directory) option so I have reworked the install part of the Imakefile
